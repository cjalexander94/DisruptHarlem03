Disrupt Harlem Code Squad



Disrupt Harlem is a code squad to help young adults learn how to code several different languages as well as offer career and entrepreneurship development. This program is based in Harlem and seeks to help African American youth become more represented in the Tech community. Disrupt harlem is run by the Harlem Business Alliance which is a community organization that seeks to fill the void of disinvestment and abandonment that has been apparent in Harlem. Among the disrupt harlem 3 cohort, students are taught by Tunisia Mitchell from the knowledge house and Chris Mercado founder of Grant Answers.
